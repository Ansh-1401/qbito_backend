## 🚀 Getting Started

### 🔧 Installation

```bash
git clone https://github.com/your-username/qbito-backend.git
cd qbito-backend
```

### ⚙️ Configure Application

```properties
spring.datasource.url=your_db_url
spring.datasource.username=your_username
spring.datasource.password=your_password
```

### ▶️ Run Application

```bash
mvn spring-boot:run
```
